# Análisis de Datos de Vehículos

Esta aplicación web permite explorar datos de vehículos de manera interactiva. Los usuarios pueden visualizar diferentes gráficos y usar controles para filtrar la información según sus necesidades.

## Funcionalidades

- Visualización de histogramas de datos de vehículos
- Creación de gráficos de dispersión interactivos
- Interfaz web intuitiva desarrollada con Streamlit

## Cómo usar la aplicación

1. Ejecuta 'streamlit run app.py'
2. Haz clic en los botones para generar diferentes visualizaciones
3. Explora los datos de manera interactiva
